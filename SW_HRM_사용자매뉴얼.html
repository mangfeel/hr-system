사회복지 인사관리시스템

직원 등록부터 급여 관리, 보고서 출력까지 — 사회복지시설 인사 업무를 위한 올인원 시스템 사용 가이드

SW HRM 저작권등록 제C-2026-007142호 2026년 2월
1
시작하기
설치 및 로그인
  1. 홈페이지에서 "다운로드" 버튼을 클릭하여 설치파일을 받습니다.
  2. 다운로드된 HRM-System-Setup-x.x.x.exe 를 실행합니다.
    Windows 보안 경고 시 → "추가 정보" → "실행" 클릭
  3. 설치 완료 후 프로그램을 실행하면 로그인 화면이 나타납니다.
  4. "회원가입" 버튼 클릭 → 이메일과 비밀번호를 입력하여 계정을 만듭니다.
  5. 관리자에게 라이선스 발급을 요청합니다.
    이메일로 승인이 완료되면 프로그램 사용이 가능합니다.
  6. 승인된 이메일과 비밀번호로 로그인하면 메인 화면이 열립니다.
권장 설정 순서

프로그램에 처음 들어오면 데이터가 비어 있습니다. 아래 순서대로 설정하면 가장 효율적입니다.

  1. 시스템 → 조직 설정
    기관 이름·주소·연락처·퇴직연금 정보를 먼저 입력합니다.
    예: 조직명 "○○종합사회복지관", 주소 "경기도 ○○시 ○○로 123", 연락처 "031-123-4567", 퇴직연금 "농협은행 DC형"
  2. 인력관리 → 신규 직원 등록
    직원 정보를 한 명씩 등록합니다. 직급·직위는 자유 텍스트로 입력하면 됩니다.
    예: 직급 "사회복지사", 직위 "팀장" 등을 직접 입력. 기존 데이터가 있으면 시스템 → 가져오기에서 엑셀로 일괄 등록도 가능합니다.
  3. 노무관리 → 급여 설정 → 직급 관리
    "직원에서 직급 불러오기" 버튼을 클릭하면 등록된 직원들의 직급이 자동으로 추출됩니다.
    예: 직원 등록 시 입력한 "사회복지사, 일반직 1급, 관리직 2급" 등이 호봉제/연봉제로 자동 분류됩니다.
  4. 노무관리 → 급여 설정 → 급여표 관리
    불러온 직급별로 호봉에 따른 기본급 금액을 입력합니다.
    예: 사회복지사 1호봉 2,127,600원, 2호봉 2,180,000원 … (인건비가이드라인 참고)
  5. 시스템 → 조직도 설정
    직원 발령 데이터에서 직위와 부서가 자동 추출됩니다. 직위 순서(관장 → 부관장 → 팀장 → 주임 …)와 부서 합치기만 설정하면 됩니다.
💡
왜 이 순서가 중요한가요?
직원 등록 시 입력한 직급·직위 정보가 급여 설정과 조직도 설정의 기초 데이터가 됩니다. 직원을 먼저 등록하면 급여 설정에서 "직급 불러오기", 조직도 설정에서 "직위/부서 자동 추출" 기능으로 입력 작업이 크게 줄어듭니다.
급여표 입력 전에 등록한 직원의 기본급은 급여표 완성 후 자동으로 반영됩니다.

왼쪽 사이드바

카테고리별 하위 메뉴가 표시됩니다. 카테고리 이름을 클릭하면 접거나 펼 수 있고, 하위 메뉴를 클릭하면 해당 기능 화면으로 이동합니다.

빠른 검색

사이드바 상단의 "빠른 검색..." 버튼을 클릭하거나 Ctrl + K 를 누르면 검색창이 열립니다. 직원 이름, 부서, 고유번호를 입력하면 해당 직원을 빠르게 찾을 수 있습니다.

3
대시보드

프로그램을 열면 가장 먼저 보이는 메인 화면입니다. 등록된 직원 데이터를 기반으로 자동 집계된 현황을 보여줍니다.

영역설명
통계 카드전체 직원, 재직 중, 휴직 중, 퇴사, 평균 호봉을 한눈에 보여줍니다. 숫자를 클릭하면 상세 목록을 확인할 수 있습니다.
빠른 실행자주 쓰는 기능(직원 등록, 인사발령, 육아휴직, 호봉획정표)으로 바로 이동하는 버튼입니다.
알림 / 예정이번 달 승급 예정자, 복직 예정 직원 등을 자동으로 알려줍니다.
이번 달 현황당월 입사자, 퇴사자, 발령 건수, 승급자 수를 표시합니다.
부서별 현황부서별 재직 인원 수를 보여줍니다. 클릭하면 소속 직원 목록이 나타납니다.
최근 활동최근에 등록·수정한 직원 이력이 시간순으로 표시됩니다.
4
인력관리
💡
직급·직위는 자유 입력
직원 등록 시 직급과 직위는 텍스트로 직접 입력합니다. 등록 후 노무관리 → 급여 설정 → 직급 관리에서 "직원에서 직급 불러오기" 버튼을 클릭하면 입력한 직급이 자동으로 추출됩니다. 이후 급여표를 입력하면 기본급이 연동됩니다.
예: 직급 "사회복지사"를 입력하고 등록 → 급여 설정에서 직급 불러오기 → 급여표에 사회복지사 1호봉 2,127,600원 입력 → 기본급 자동 반영
신규 직원 등록

새 직원을 시스템에 등록하는 화면입니다. 3개 카드로 구성되며, * 표시 항목은 반드시 입력해야 합니다.

카드 1 — 기본 정보
항목입력 방법
고유번호자동시스템이 자동 부여합니다 (H001, H002 …). 수정 불가.
발령 코드도 이 번호 기준으로 생성됩니다. 예: H105-01(첫 번째 발령), H105-02(두 번째 발령)
사원번호선택기관 자체 사원번호. 입사일 입력 시 자동 생성되며, 🔄 버튼이나 직접 수정도 가능.
예: 2026년 입사 3번째 직원 → 2026-0003 자동 생성
성명필수직원 이름. 예: 김민수
입사일필수입사 날짜 선택. 이 날짜 기준으로 호봉과 근속기간이 자동 계산됩니다.
예: 2024-03-02 입사, 과거 경력 3년 인정 → 입사호봉 4호봉 결정
고용형태필수정규직 / 무기계약직 / 계약직 / 육아휴직대체 중 선택
육아휴직대체: 기존 직원의 육아휴직 기간 동안 대체 근무하는 직원
카드 2 — 소속 및 급여 정보
항목입력 방법
부서필수소속 부서명. 예: 사무국, 사회복지팀, 주간보호팀, 장애인활동지원팀
직위필수직위. 예: 관장, 부장, 팀장, 주임, 사원
조직도 표시 순서와 직책수당에 영향을 줍니다.
직급필수인건비가이드라인 기준 직급. 예: 사회복지사, 일반직 1급, 관리직 2급, 선임사회복지사
⚠ 이 직급이 급여표와 연동되어 기본급이 결정됩니다. 급여 설정 → 직급 관리에서 먼저 등록한 직급명과 정확히 일치해야 합니다.
직종필수자격 기반 직종. 예: 사회복지사, 요양보호사, 간호사, 영양사, 조리원
직급과 직종은 다를 수 있습니다. 예: 직급 "일반직 1급" + 직종 "사무원"
주 소정근로시간1주일 근로시간 입력 (기본 40시간). 단축근로자는 해당 시간 입력.
예: 주 40시간 → 월 209시간 / 주 35시간 → 월 182.87시간 / 주 15시간(시간제) → 월 78.37시간
월 소정근로시간자동주 소정근로시간을 기반으로 자동 계산됩니다. (주간 시간 × 4.345주)
급여체계필수호봉제 또는 연봉제 선택
호봉제: 직급별 급여표에 따라 기본급 결정 / 연봉제: 별도 연봉 책정
카드 3 — 과거 경력 (선택)

입사 전 동종 기관 경력이 있으면 입력합니다. 경력이 없으면 "경력 없음(신입)" 체크하면 됩니다. 입력한 경력은 호봉 계산에 자동 반영됩니다.

항목입력 방법
근무처/경력 내용이전 근무기관. 예: ○○주간보호센터, △△노인복지관, □□장애인복지관
시작일 / 종료일해당 기관에서의 근무 기간. 예: 2018-03-01 ~ 2022-02-28 (4년)
인정률 (%)경력 인정 비율 (0~100%). 기관 규정에 따라 입력.
동종(사회복지관→사회복지관): 100% / 유사(노인요양→사회복지관): 80% / 이종: 기관 규정
주당근무시간해당 기관에서의 주 근로시간 (1~40). 풀타임이 아닌 경우 비례 환산됩니다.
예: 실제 4년 근무, 주 20시간(파트타임) → 환산 경력 2년
💡
등록 예시 시나리오
김민수 사회복지사가 2026-03-02에 입사합니다.
① 기본 정보: 성명 "김민수", 입사일 2026-03-02, 정규직
② 소속/급여: 사회복지팀, 주임, 직급 "사회복지사", 직종 "사회복지사", 주 40시간, 호봉제
③ 과거 경력: △△노인복지관 2022-03-02~2025-02-28 (3년), 인정률 100%, 주 40시간
→ "호봉 계산 및 등록" 클릭 → 과거경력 3년 반영 → 입사호봉 4호봉 자동 산출, 기본급 2,234,700원 결정
"경력 추가" 버튼으로 여러 건의 경력을 입력할 수 있습니다.
📋
실전 예시 — 경력직 사회복지사 등록

상황: 김사회(사회복지사 1급)가 2026년 3월 2일 ○○종합사회복지관에 입사합니다. 이전에 △△주간보호센터에서 3년(2022-03-01 ~ 2025-02-28), □□노인복지관에서 2년(2020-03-01 ~ 2022-02-28) 근무한 경력이 있습니다.

입력 방법:
① 카드1: 성명 "김사회", 입사일 "2026-03-02", 고용형태 "정규직"
② 카드2: 부서 "사회복지팀", 직위 "사원", 직급 "사회복지사", 직종 "사회복지사", 급여체계 "호봉제"
③ 카드3: 경력1 — □□노인복지관, 2020-03-01 ~ 2022-02-28, 인정률 100%, 주당 40시간
　　　　 경력2 — △△주간보호센터, 2022-03-01 ~ 2025-02-28, 인정률 100%, 주당 40시간

결과: "호봉 계산 및 등록" 클릭 → 환산 총 경력 5년, 입사호봉 6호봉, 기본급 2,386,400원 자동 산출
직원 목록/수정

등록된 직원을 조회하고 정보를 수정하는 화면입니다. 상단에 상태별 통계 카드(전체/재직/퇴사/휴직중)가 표시되며, 클릭하면 해당 상태로 필터링됩니다.

조회 기능
기능설명
상태 필터상단 통계 카드(전체/재직/퇴사/휴직중) 클릭 시 해당 상태만 필터링. 예: "휴직중" 클릭 → 육아휴직 중인 직원만 표시
검색이름, 부서, 고유번호로 실시간 검색. 예: "복지팀" 입력 → 사회복지팀 소속 직원만 표시
부서 필터드롭다운에서 특정 부서만 선택하여 보기. 예: "주간보호팀" 선택

급여 설정

급여 계산에 필요한 각종 기준을 연도별로 설정합니다. 6개 탭으로 구성됩니다.

탭 1 — 직급 관리

호봉제 직급과 연봉제 직급을 관리합니다. 직원 등록 시 입력한 직급을 "직원에서 직급 불러오기" 버튼으로 자동 추출하거나, 직접 등록할 수 있습니다.

구분예시
호봉제 직급사회복지사, 일반직 1급, 관리직 2급, 선임사회복지사, 생활지도원 등
인건비가이드라인의 급여표 직급명과 동일하게 입력합니다. 직급명이 한 글자라도 다르면 급여 연동이 안 됩니다.
연봉제 직급영양사, 조리원, 운전원, 시간제 등
호봉표가 적용되지 않는 직급입니다. 별도 연봉 계약으로 관리합니다.

실전 예시 — 30인 규모 종합사회복지관의 직급 설정

방법 1 — 직원에서 불러오기 (권장):
직원 등록을 먼저 완료한 경우 → "2026년 근무 직원에서 직급 불러오기" 버튼 클릭 → 등록된 직원의 직급이 자동 추출되어 호봉제/연봉제로 분류

방법 2 — 직접 등록:
호봉제: "사회복지사" → "선임사회복지사" → "일반직 1급" → "관리직 2급" → "생활지도원"
연봉제: "영양사" → "조리원" → "운전원"

* 직급 이름은 시·도별 인건비가이드라인의 급여표 항목명과 완전히 일치해야 합니다.
탭 2 — 급여표 관리

직급별·호봉별 기본급을 입력하는 표입니다. 연도를 선택하고, 최대 호봉 수를 설정(예: 30호봉)한 뒤, 각 칸에 원 단위 금액을 입력합니다.

항목설명
연도필수급여표가 적용되는 연도 선택. 예: 2026년 → 2026년 인건비가이드라인 금액 입력
최대 호봉급여표의 최대 호봉 수 설정. 예: 30호봉 (기관 규정에 따라)
전년도 복사작년 급여표를 가져와서 변경분만 수정. 매년 인건비가이드라인 변경 시 유용
💡
입력 예시 (2026년 인건비가이드라인 기준)
사회복지사: 1호봉 2,127,600원 / 2호봉 2,180,000원 / 3호봉 2,234,700원 / 4호봉 2,291,400원 / 5호봉 2,350,200원 …

탭 3 — 직책수당 설정

직위(직책)별 월 수당 금액을 설정합니다. 직위를 선택(또는 직접 입력)하고 금액을 입력한 뒤 "추가"합니다.

💡
입력 예시
관장 300,000원 / 부관장 250,000원 / 부장 200,000원 / 팀장 150,000원 / 주임 100,000원 / 사원 0원
직위명이 직원 등록·발령에서 사용하는 직위명과 정확히 일치해야 자동 적용됩니다.
예: 급여현황표에서 김민수(사회복지팀/팀장)의 직책수당에 150,000원이 자동 표시됩니다.

항목입력 방법

탭 5 — 통상임금 설정

시간외근무수당 계산 기초가 되는 통상임금에 포함할 항목을 설정합니다.

항목설명
명절휴가비 포함체크 시 통상임금에 포함. 산정 방식 선택 가능:
연간 고정: (설 기본급×비율 + 추석 기본급×비율) ÷ 12 → 매월 동일 금액
월별 연동: 해당 월 기본급 × (비율 합계 ÷ 12) → 호봉 변동 시 자동 연동
직책수당 포함체크 시 통상임금에 포함. 예: 팀장 직책수당 150,000원이 통상임금에 합산
직무대리 직책수당 포함체크 시 겸직/직무대리의 직책수당도 통상임금에 포함
📋
계산 예시 — 팀장 김민수의 통상임금

기본급 2,498,000원(사회복지사 8호봉) + 직책수당 150,000원(팀장) + 명절휴가비 월할액 249,800원
= 통상임금 2,897,800원
→ 통상시급: 2,897,800원 ÷ 209시간 = 13,865원 (시간외수당 계산 기초)

교차 통계 분석기준일을 선택하고 부서별·직급별·직위별·직종별·고용형태별 등으로 교차 분석합니다. 표와 그래프로 결과 표시.
빠른 분석 (원클릭): 부서별 인원현황, 직위별 인원현황, 직종별 현황, 부서×직위 교차분석 등 자주 쓰는 통계 즉시 생성
예: "부서×직위 교차분석" 클릭 → 사회복지팀(팀장 1, 주임 1, 사원 2) / 주간보호팀(팀장 1, 생활지도원 3) / 사무국(부장 1, 주임 1) 교차표 자동 생성
근속현황표재직 직원의 근속연수별 분포(1년 미만, 1~3년, 3~5년, 5~10년, 10년 이상)를 보여줍니다.
연명부기준일에 재직 중이었던 모든 직원의 기본 정보를 표로 출력합니다. 24개 항목 중 필요한 것만 선택 가능. 프리셋(간략 5개/기본 12개/상세 전체) 지원. 엑셀 다운로드 및 A4 세로/가로 인쇄.
예: 기준일 "2026-03-01" + 기본 프리셋 → 고유번호, 성명, 부서, 직위, 직급, 입사일, 호봉, 근속기간 등 12개 항목 표 생성
호봉획정표직원의 호봉 산출 근거를 상세히 보여줍니다. 3가지 출력 양식(공문서/모던/표준), 직원 검색·필터 후 일괄 생성 가능.
예: 김사회 호봉획정표 → 과거경력 5년 + 현 기관 3년 = 총 경력 8년, 현재 8호봉 근거 문서 출력
입사자 목록시작일~종료일 기간 동안 입사한 직원 목록. 컬럼 선택, 프리셋, 엑셀 다운로드, 인쇄 가능.
예: 2026-01-01 ~ 2026-12-31 → 올해 입사한 직원 3명 목록 출력
퇴사자 목록시작일~종료일 기간 동안 퇴사한 직원 목록. 20개 컬럼 중 선택, 엑셀 다운로드, 인쇄 가능.

시스템
조직 설정

기관의 기본 정보를 설정합니다. 근로계약서, 인쇄물 등에 자동 반영됩니다.

항목입력 방법
조직명필수기관의 정식 명칭. 예: ○○종합사회복지관
기관 주소기관 소재지 주소. 예: 경기도 ○○시 ○○로 123
기관 연락처대표 전화번호. 예: 031-123-4567
퇴직연금 은행농협, 신한, 국민, 우리, 하나, 기업, SC제일, 한국투자증권, 삼성생명, 미래에셋증권 중 선택
퇴직연금 유형DC형(확정기여형) 또는 DB형(확정급여형). 예: 농협은행 DC형
조직도 설정

보고서 > 조직도의 표시 방식을 설정합니다.

기능설명
직위 순서직위의 표시 순서를 설정합니다. 각 직위에 역할(기관장/부기관장/부서장/팀원)을 지정합니다.
예: 관장(기관장) → 부관장(부기관장) → 부장(부서장) → 팀장(부서장) → 주임(팀원) → 사원(팀원)
부서 합치기조직도에서 소규모 부서를 하나로 묶어 표시할 수 있습니다.
예: "경리팀"과 "총무팀"을 "사무국"으로 합쳐서 표시
비고란 역할 표시조직도 비고란에 기관장/부기관장/부서장 역할을 표시합니다.
겸직/직무대리 관리

직원이 다른 부서를 겸직하거나 타 직원의 직무를 대리하는 경우를 등록합니다.

유형설명
겸직직원이 현재 소속 외 다른 부서 업무를 겸임하는 경우. 직원·겸직 부서·직위·기간 입력.
예: 사무국 부장이 사회복지팀장을 겸직 (2026-03-01 ~ 별도 종료일 없음)
직무대리특정 직위의 업무를 타 직원이 대리하는 경우. 직원·대리 부서·직위·기간 입력.
예: 팀장 육아휴직 기간 중 주임이 팀장 직무대리 (2026-04-01 ~ 2027-03-31)

등록된 겸직/직무대리는 조직도에 자동 반영되며, 통상임금 설정에서 "직무대리 직책수당 포함"을 체크하면 급여 계산에도 반영됩니다.

백업/복원

시스템 데이터를 안전하게 보관하고 복원하는 기능입니다. 정기적인 백업을 강력히 권장합니다.

기능설명
HRM 백업
권장		 "HRM 백업" 버튼 클릭 시 .hrm 파일이 자동 저장됩니다. 직원 정보, 발령 이력, 급여 설정, 포상 데이터 등 모든 데이터가 보안 인코딩되어 완벽하게 보존됩니다.
엑셀 백업 엑셀 형식으로 저장. 다른 프로그램에서 데이터 확인 시 유용. 일부 설정이 포함되지 않을 수 있습니다.
시스템 복원 "백업 파일 선택" 클릭 → 이전에 저장한 .hrm 또는 .json 파일을 선택하면 복원됩니다.
전체 삭제 모든 데이터를 삭제합니다. 되돌릴 수 없으므로 반드시 백업 후 실행하세요.
🚨
복원하면 현재 데이터가 모두 대체됩니다. 복원 전에 현재 데이터를 먼저 백업해두세요.
가져오기

다른 곳에서 관리하던 데이터를 엑셀로 일괄 등록합니다. 처음 시작 시 기존 데이터를 옮길 때 유용합니다.

기능설명
인사 기초데이터 엑셀 파일(.xlsx, .xls) 선택 → 직원 정보, 발령 이력 자동 등록.
※ 과거 경력은 포함되지 않습니다. 완벽한 이관은 HRM 백업을 사용하세요.
포상 데이터 기존 포상대장 엑셀을 업로드합니다. 성명+입사일로 직원 DB와 자동 매칭됩니다.
컬럼 순서: 포상구분 | 성명 | 입사일 | 퇴사일 | 퇴사여부 | 직위 | 포상연도 | 포상내역 | 훈격 | 포상주관처 | 포상내용 | 선정여부 | 수상년월일 | 상장사진
프로그램 정보

프로그램 버전, 저작권 등록 정보, 라이선스 정보를 확인할 수 있습니다.

자주 묻는 질문
직원의 호봉은 어떻게 계산되나요?
입사일, 과거 경력(인정비율 적용), 주당근로시간 등을 종합하여 자동 계산됩니다. "인력관리 → 신규 직원 등록"에서 정보를 입력하고 "호봉 계산 및 등록" 버튼을 누르면 결과를 확인할 수 있습니다.
호봉표를 매년 바꿔야 하나요?
네, 매년 인건비가이드라인이 변경되므로 "노무관리 → 급여 설정 → 급여표 관리"에서 해당 연도의 급여표를 새로 입력해주세요. 연도별로 별도 관리됩니다.
직원이 부서를 이동하면 어떻게 하나요?
"인력관리 → 인사 발령"에서 발령을 등록하세요. 발령 등록 후 직원의 부서, 직위 등이 자동으로 변경됩니다.
퇴사 처리는 어떻게 하나요?
"인력관리 → 직원 목록/수정"에서 해당 직원을 클릭한 후 "퇴사 처리" 버튼을 누르고 퇴사일을 입력하면 됩니다.
데이터가 날아갈까 걱정됩니다.
"시스템 → 백업/복원"에서 정기적으로 HRM 백업을 해주세요. 백업 파일(.hrm)을 USB, 클라우드 등 안전한 곳에 보관하면 언제든 복원할 수 있습니다.
여러 대의 컴퓨터에서 같은 데이터를 쓸 수 있나요?
현재 버전에서는 데이터가 각 PC에 개별 저장됩니다. 한 PC에서 HRM 백업한 파일을 다른 PC에서 복원하면 데이터를 옮길 수 있습니다.
근로계약서에 기관장 이름이 안 나옵니다.
"시스템 → 조직 설정"에서 조직명이 입력되어 있는지 확인하세요. 기관장 정보는 등록된 직원 중 가장 높은 직위의 직원이 자동 표시됩니다. "시스템 → 조직도 설정"에서 직위 순서를 확인해주세요.
급여 현황표의 금액이 이상합니다.
"노무관리 → 급여 설정"에서 해당 연도의 급여표 금액, 직책수당 설정, 수당 계산 설정(단수처리 방식)이 올바르게 되어 있는지 확인하세요.
프로그램 업데이트는 어떻게 하나요?
새 버전 출시 시 프로그램 실행 시 자동으로 업데이트 알림이 나타납니다. 안내에 따라 업데이트하면 됩니다.
문의는 어디로 하나요?
개발일지 블로그 swhrm.tistory.com 을 통해 문의하실 수 있습니다.
